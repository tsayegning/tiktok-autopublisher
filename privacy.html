Politique de confidentialité

Auto TikTok Publisher

1. Données collectées

Cette application ne collecte, ne stocke et ne traite aucune donnée personnelle d'utilisateurs tiers. Elle est utilisée uniquement par son propriétaire.

2. Utilisation de l'API TikTok

L'application se connecte au compte TikTok du propriétaire via les mécanismes d'authentification fournis par TikTok (OAuth). Les jetons d'accès nécessaires au fonctionnement de l'application sont utilisés uniquement pour :

  • téléverser des vidéos sur le compte TikTok du propriétaire ;
  • publier ces vidéos ;
  • obtenir des informations basiques sur le compte connecté.

3. Partage de données

Aucune donnée n'est vendue, transférée ou partagée avec des tiers. Toutes les interactions se font directement entre l'application et les serveurs de TikTok.

4. Sécurité

Le propriétaire de l'application prend les mesures nécessaires pour protéger les informations d'authentification (par exemple : jetons d'accès) et éviter tout accès non autorisé.

5. Contact

Pour toute question concernant cette politique de confidentialité, veuillez contacter le propriétaire de l'application via les moyens habituels (par exemple, email personnel).
